# DermaSmart Mobile App 📱

DermaSmart is a cross-platform mobile application built with Flutter that offers users personalized skincare routines through skin analysis and recommends products suited to their needs.

**Our Vision:** *"A skin expert in everyone's pocket."*

## ✨ Key Features

* **Personalized Routines:** A smart system that creates separate, step-by-step morning and evening routines based on the user's specific concerns and skin type.
* **Ingredient Conflict Rule Engine (Rule-Based Engine):** A rule-based infrastructure that prevents ingredient conflicts such as niacinamide, glycolic acid, and vitamin C, ensuring safe product use.
* **Skin Type and Problem Analysis:** Target-focused active ingredient recommendations by identifying problems such as acne, dryness, or blemishes.

- ## 🚀 Frontend (Mobile) Technologies
- **Framework:** Flutter
- **Programming Language:** Dart
- **State Management:** Riverpod
- **Routing:** GoRouter
- **Networking:** HTTP Package
- **Local Data Storage:** Shared Preferences
- **UI and Design Packages:** Material Design, Google Fonts, Table Calendar
- **Native Integrations:** Flutter Native Splash, Flutter Launcher Icons
- **Backend Connection:** REST API via ASP.NET Core Web API (.NET 8)

## 🛠️ Installation and Running Steps

Follow the steps below in order to build the project on your own computer and run it on an emulator/physical device.

### 1️⃣ Download Dependencies
To install all the packages used in the project:
```bash
flutter pub get
```

---

### 2️⃣ Generate Splash Screen
To generate the app's splash screen:
```bash
dart run flutter_native_splash:create
```

---

### 3️⃣ Generate App Icon
To generate the launcher icons:
```bash
dart run flutter_launcher_icons
```

---

### 4️⃣ Run the App
To run the project on an emulator or physical device:
```bash
flutter run
```

---

## 📦 Building a Release APK
To build an APK in release mode:
```bash
flutter build apk --release
```
